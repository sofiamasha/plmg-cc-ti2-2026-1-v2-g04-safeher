SafeHer para sites de vagas

Integre o score de segurança do SafeHer no seu site de vagas em 5 minutos. Mostre às candidatas a reputação da empresa antes de elas se candidatarem.

O que é?

A API Pública SafeHer permite que qualquer site de vagas (Gupy, Catho, sites internos de RH, etc.) consulte o score que mulheres atribuíram a uma empresa e exiba um badge de reputação nas vagas.

Exemplo do badge que seu site pode mostrar: ⭐ SafeHer 4.3 · 27 avaliações

🎯 Demonstração ao vivo

📡 Endpoints públicos

    
Base URL: http://localhost:3000/api/public

    
Envie a chave em X-API-Key: <sua_chave> ou ?api_key=<sua_chave>.


    

      
        
MétodoRotaDescrição

      
      
        

          GET
          /empresas/score-por-nome?nome=X
          Atalho perfeito para sites de vagas: passe o nome direto da página da vaga e receba o score. Devolve encontrada: false se a empresa ainda não tem dados.
        

        

          GET
          /empresas/buscar?nome=X
          Lista até 5 empresas que combinam com o nome, com score e qtd de avaliações.
        

        

          GET
          /empresas/:id/score
          Score detalhado de uma empresa específica (por ID), incluindo os motivos mais citados.

🔌 Como integrar (3 passos)

    
    
      
  1. 
        Solicite sua chave: mande um e-mail para
        parceiros@safeher.local com nome do site e contato.
        Você receberá uma chave no formato sh_live_xxxxxxxx.
      
    2. 
      
  2. 
        Adicione o snippet abaixo na página da vaga, substituindo
        SUA_CHAVE e o nome da empresa:

    <script>
fetch('http://localhost:3000/api/public/empresas/score-por-nome?nome='
    + encodeURIComponent('Empresa X')
    + '&api_key=SUA_CHAVE')
  .then(r => r.json())
  .then(d => {
    if (d.encontrada && d.score > 0) {
      document.getElementById('safeher-badge').innerHTML =
        '⭐ SafeHer ' + d.score + ' · ' + d.quantidadeAvaliacoes + ' avaliações';
    }
  });
</script>
<span id="safeher-badge"></span>
    
      
    3. 
      
  3. 
        Pronto. O badge aparece automaticamente em toda página
        de vaga que tiver dados no SafeHer.

🛡️ Limites e segurança

    
    
      
  • Rate limit: 1000 chamadas/dia por chave (configurável).
    • 
      
  • Apenas leitura: endpoints públicos não permitem criar, alterar ou excluir nada.
    • 
      
  • Dados agregados: retornamos média e contagem — nunca expomos identidade das avaliadoras.
    • 
      
  • Headers de quota: X-RateLimit-Limit e X-RateLimit-Remaining em toda resposta.
