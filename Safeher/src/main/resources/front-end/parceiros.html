O que é a API SafeHer?

A API Pública SafeHer permite que qualquer site de vagas — Gupy, Catho, portais internos de RH — consulte o score que mulheres atribuíram a uma empresa e exiba um badge de reputação nas vagas, ajudando candidatas a tomar decisões mais informadas.

Badge que seu site pode exibir:
SafeHer 4.3 · 27 avaliações

Endpoints públicos

                

                

                    Envie a chave em X-API-Key: <sua_chave> ou ?api_key=<sua_chave>.
                

                

                    
                        

                            Método
                            Rota
                            Descrição
                        

                    
                    
                        

                            GET
                            /empresas/score-por-nome?nome=X
                            Atalho ideal para sites de vagas. Passe o nome da empresa e receba o score. Retorna encontrada: false se ainda não há dados.
                        

                        

                            GET
                            /empresas/buscar?nome=X
                            Lista até 5 empresas que combinam com o nome, com score e quantidade de avaliações.
                        

                        

                            GET
                            /empresas/:id/score
                            Score detalhado por ID, incluindo os motivos mais citados pelas avaliadoras.

Como integrar (3 passos)

                

                
    
                    
  1. 
                        Solicite sua chave de acesso
                        
    Envie um e-mail para parceiros@safeher.local com o nome do site e contato. Você receberá uma chave no formato sh_live_xxxxxxxx.
    
                    
    2. 
                    
  2. 
                        Adicione o snippet na página da vaga
                        
    Substitua SUA_CHAVE e o nome da empresa. O badge aparece automaticamente quando há dados:
    

    // Adicione na página da vaga
fetch('/api/public/empresas/score-por-nome?nome='
    + encodeURIComponent('Nome da Empresa')
    + '&api_key=SUA_CHAVE')
  .then(r => r.json())
  .then(d => {
    if (d.encontrada && d.score > 0) {
      document.getElementById('safeher-badge').innerHTML =
        '⭐ SafeHer ' + d.score + ' · ' + d.quantidadeAvaliacoes + ' avaliações';
    }
  });

<span id="safeher-badge"></span>
    
                    
    3. 
                    
  3. 
                        Pronto!
                        
    O badge exibe o score real das avaliações de mulheres sempre que a empresa tiver dados na SafeHer.
    
                        
    
                            Resultado:
                            
    
                                
                                SafeHer 4.3 · 27 avaliações

Limites e segurança

                

                

                    

                        

                        

                            
Rate limit

                            
1.000 chamadas/dia por chave, configurável para planos de parceiros.

                        

                    

                    

                        

                        

                            
Apenas leitura

                            
Os endpoints públicos não permitem criar, alterar ou excluir nenhum dado.

                        

                    

                    

                        

                        

                            
Dados agregados

                            
Retornamos apenas médias e contagens — nunca a identidade das avaliadoras.

                        

                    

                    

                        

                        

                            
Headers de quota

                            
X-RateLimit-Limit e X-RateLimit-Remaining em toda resposta.
